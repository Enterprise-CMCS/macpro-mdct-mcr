# MDCT-MCR playwright reports

This branch exists to house playwright reports and display them using [GitHub Pages](https://pages.github.com/).

Older reports are deleted nightly with the [delete-pages workflow](https://github.com/Enterprise-CMCS/macpro-mdct-mcr/blob/main/.github/workflows/delete-pages.yml).

For more about this application, please view the [`main` branch README.md](https://github.com/Enterprise-CMCS/macpro-mdct-mcr/blob/main/README.md).
